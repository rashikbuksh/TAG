Can not get param with "/"

Description

If any parameter contains "/", then add "(*)" after the parameter name. For example, in the following route: "/change/job-entry/:order_number(*)" the parameter "order_number" represents a value like "Anik-13/3/2023-Paharika-Srilanka"

Changes

Put "(*)" after the parameter name.
